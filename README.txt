En este proyecto estaré realizando una búsqueda por medio de filtraciones

Para esto, realice una web de carros, donde tendra la marca, year, precio mínimo, precio máximo, puertas, transmisión y color.

Cuando el usuario seleccione una de las opciones mostradas en cada uno de los campos, nuestro web realizará una búsqueda estricta, 
para filtrar dichos valores de los campos seleccionados.

Este proyecto lo realice con HTML, CSS3 y JavaScript.
